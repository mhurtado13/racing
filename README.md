RaCInG
================

The random graph model to infer cell-cell communication networks based
on bulk RNA-seq data.

## Installation

You can install the development version of `racing` from GitHub with:

``` r
# install.packages("devtools")
devtools::install_github("mhurtado13/racing")
```

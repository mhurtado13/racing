# Read a RaCInG simulation output file

Read a RaCInG simulation output file

## Usage

``` r
Read_Sim_Output(filename)
```

## Arguments

- filename:

  Path to a `.out` file generated by
  [`runSim()`](https://mhurtado13.github.io/racing/reference/runSim.md).

## Value

A list containing the average values, standard deviations, and raw
counts.
